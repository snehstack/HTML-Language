# 🚀 CryptoM5 – Cryptocurrency Landing Page

A responsive cryptocurrency landing page developed using **HTML5**. This project showcases a complete multi-section website for a crypto trading platform, including navigation, feature cards, pricing plans, traders, articles, contact form, Google Maps integration, and more.

---

## ✨ Features

* 🏠 Home Section
* 📖 About Us
* ⭐ Features Section
* 👨‍💼 Our Traders
* 📊 Skills Progress
* 💰 Pricing Plans
* 🏆 Top Traders
* 🎥 Video Section
* 📰 Articles Section
* 🤝 Our Clients
* 📩 Contact Form
* 🗺️ Google Maps Integration
* 📱 Simple and Clean Layout

---

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Ensure all images and videos are inside their respective folders.
4. Open `index.html` in your preferred web browser.
